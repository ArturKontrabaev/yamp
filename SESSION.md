# YAMP — Yandex Music Player (menubar app)

## Последнее обновление
2026-03-29 — проект создан

## Контекст
macOS menubar-приложение на Swift. Показывает текущий трек Яндекс Музыки в menu bar.
Подход: System Now Playing (MediaRemote) → показ в NSStatusItem.
